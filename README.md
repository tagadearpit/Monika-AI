## 🔥 **What is Monika-AI?**

**Monika-AI** isn't just another chatbot. It's a **personality-driven, vision-enabled AI companion** that understands context, remembers conversations, and responds with charm and intelligence. Powered by **Google's Gemini 2.5 Flash**, Monika delivers natural conversations with stunning graphics, voice capabilities, and a presence that feels almost real.

Whether you're looking for witty banter, practical assistance, or just someone to talk to—**Monika is here**, and she's absolutely captivating.

### ⚡ **Why Choose Monika-AI?**

✅ **Real-Time Vision Recognition** — Show Monika objects, code, documents. She analyzes instantly.  
✅ **Voice & Speech Synthesis** — Hands-free interaction with dynamic voice modulation.  
✅ **Multi-User Memory System** — Secure, isolated conversations with persistent memory.  
✅ **Beautiful UI/UX** — Glassmorphism design with dynamic themes (Midnight, Rose, Cyber, Normal).  
✅ **Progressive Web App** — Install on any device. Works offline. Native app feel.  
✅ **Enterprise Security** — XSS protection, rate limiting, input validation, encrypted OTPs.  
✅ **Picture-in-Picture Window** — Keep Monika floating while you work.  
✅ **Multiple Auth Methods** — Google OAuth, Firebase Phone Auth, Email OTP.

## 🚀 **Quick Start Guide**

### 📋 **Prerequisites**
- **Node.js** ≥ 18.0.0 (Get it from [nodejs.org](https://nodejs.org/))
- **npm** or **yarn** (Comes with Node.js)
- **MongoDB Atlas** account ([Create free](https://www.mongodb.com/cloud/atlas))
- **Google Gemini API Key** ([Get free here](https://ai.google.dev/))
- **Firebase Project** ([Setup free](https://firebase.google.com/))
- **Brevo Account** (for email, [optional & free](https://www.brevo.com/))

### 🎯 **Installation**

**Step 1: Clone the Repository**
```bash
git clone https://github.com/tagadearpit/Monika-AI.git
cd Monika-AI/backend
```

**Step 2: Install Dependencies**
```bash
npm install
```

**Step 3: Configure Environment Variables**

Create a `.env` file in the `/backend` directory:

```env
# 🔹 Core Configuration
PORT=10000
NODE_ENV=development

# 🔹 AI & Database
GEMINI_API_KEY=your_gemini_api_key_here
GEMINI_API_KEY_2=your_backup_gemini_key_here
MONGO_URI=mongodb+srv://username:password@cluster.mongodb.net/monika-db

# 🔹 CORS & Security
ALLOWED_ORIGINS=http://localhost:10000,https://monika-ai-0jpf.onrender.com

# 🔹 Firebase Authentication
FIREBASE_API_KEY=your_firebase_api_key
FIREBASE_AUTH_DOMAIN=your-project.firebaseapp.com
FIREBASE_PROJECT_ID=your-project-id
FIREBASE_STORAGE_BUCKET=your-project.appspot.com
FIREBASE_MESSAGING_SENDER_ID=your-sender-id
FIREBASE_APP_ID=your-app-id

# 🔹 Google OAuth
GOOGLE_CLIENT_ID=your_google_client_id.apps.googleusercontent.com

# 🔹 Email Service (Brevo SMTP)
SMTP_USER=your-brevo-email@example.com
SMTP_PASS=your-brevo-api-key
SMTP_FROM_EMAIL=noreply@monika-ai.com
```

**Step 4: Run Locally**

**Development Mode:**
```bash
npm run dev
```

**Production Mode:**
```bash
npm run prod
```

**Step 5: Open in Browser**
```
http://localhost:10000
```

> **💡 Tip:** Use Chrome or Chromium-based browsers for full Vision, Speech, and Picture-in-Picture support.

## 📡 **API Reference**

### 🔐 **Authentication Endpoints**

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/auth/send-otp` | POST | Send OTP code via email |
| `/api/auth/verify-otp` | POST | Verify email OTP |
| `/api/config` | GET | Fetch Firebase & Google configuration |

### 💬 **Chat & Memory Endpoints**

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/ask` | POST | Send message with optional image (rate limited) |
| `/api/history/:sessionId` | GET | Retrieve chat history for session |
| `/api/facts/:sessionId` | GET | Get learned personal facts |

### 📊 **Request/Response Examples**

**Send a Message with Image:**
```bash
curl -X POST http://localhost:10000/ask \
  -H "Content-Type: application/json" \
  -d '{
    "sessionId": "user-uuid",
    "message": "What is this?",
    "imageData": "data:image/jpeg;base64,...",
    "timestamp": 1234567890
  }'
```

**Get Chat History:**
```bash
curl http://localhost:10000/api/history/user-uuid
```

## 🐛 **Troubleshooting**

### Vision API Not Working?
- Ensure your browser has camera permissions
- Check that Gemini API key is valid
- Verify CORS settings in `.env`

### Voice Not Recognized?
- Grant microphone permissions in browser
- Test microphone in browser settings
- Check that browser supports Web Speech API

### Can't Login?
- Verify Firebase configuration is correct
- Check that CORS origins include your domain
- Clear browser cache and try again

### Slow Responses?
- Check Gemini API rate limits
- Verify MongoDB connection is stable
- Reduce conversation history length
- Check network latency
